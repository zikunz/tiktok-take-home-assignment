# 抖音电子商城

这是一个使用 React 构建的抖音电商应用。用户可以浏览产品，将其添加到购物车中并模拟下单。

## 主要功能

- 浏览产品列表
- 添加产品到购物车
- 查看购物车并调整商品数量
- 支持中英文切换

## 技术栈

- React
- Ant Design
- React Router
- Jotai
- Styled-components
- React Intl

## 安装和运行

1. 克隆项目到本地：

   ```bash
   git clone https://github.com/zikunz/tiktok-take-home-assignment.git
   cd tiktok-take-home-assignment
   ```

2. 安装依赖：
   ```bash
   npm install
   ```
3. 运行项目：
   ```bash
   npm start
   ```

## 部署
